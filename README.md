Hier toon ik mijn projecten, skills en contactinformatie.

## ✦ Technologieën

- React
- Vite
- JavaScript
- HTML
- Tailwind CSS en custom CSS
- React Router
- Git en Github

## ✦ Functies

- Over mij sectie
- Projecten overzicht
- Informatie per project

## ✦ Installatie en gebruik

1. Clone de repository:

```sh
git clone https://github.com/eline-vanstraten/eline-vanstraten.github.io.git
```

2. Ga naar de map:

```sh
cd eline-vanstraten.github.io
```

3. Installeer dependencies:

```sh
npm install
```

4. Start de development server:

```sh
npm run dev
```
